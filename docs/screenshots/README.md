# Screenshot checklist

Capture these views from the running app and save them beside this file before publishing the repository:

1. `catalog.png` — the bull catalog with filters visible.
2. `supplier-dashboard.png` — the supplier inventory dashboard.
3. `payment.png` — the payment/receipt view with no personal information visible.

Use a consistent browser size (for example 1440 × 900), remove browser chrome, and avoid credentials, API URLs, or location details in captures.
